# Southern Paiute Center Website

Static HTML website for the Southern Paiute Center.

## Deployment

This site can be deployed to Vercel for reliable access:

1. Install Vercel CLI: `npm i -g vercel` (or use `npx vercel`)
2. Run `vercel` in this directory
3. Follow prompts to link to your Vercel account
4. Site will be available at `southern.vercel.app` or your custom domain

Alternatively, deploy via Vercel dashboard:
- Go to vercel.com
- Import GitHub repo `parkphillip/southern`
- Vercel will auto-detect and deploy

For custom domain `southernpaiute.org`:
- Add domain in Vercel project settings
- Update DNS records as instructed by Vercel
